Programming paradigm | 编程范式

根据编程语言的特点对编程语言进行分类的一种方法


常见的范式:

命令式, 声明式, 面向对象, 函数式

命令式

指明操作发生的顺序，显式控制顺序的构造，允许副作用。

声明式

没有指明操作执行的顺序，描述期望得到的结果。

例如 sql, html

面向对象

使用由数据字段和方法组成的数据结构及其交互（对象）来设计程序。

大多数面向对象语言也是命令式语言。

函数式

使用数学函数的组合，并避免副作用。

大多数函数式语言也是声明式语言。

现代语言大多都是多范式的

考虑到没有一种范式能以最简单或最有效的方式解决所有问题

这样的语言的设计目标是允许程序员为一个给定的工作

使用最合适的编程风格和相关的语言结构
